A seasoned molecular biologist with a passion for data-driven decision-making

Throughout my career in biotechnology, I've had the privilege of collaborating with a diverse team of talented scientists, engineers, and software developers.

Encountering the unexpected is part of working in cutting-edge research. I enjoy having the opportunity to innovate and problem-solve, often picking up new skills in the process.

I quickly realised that improving my data manipulation skills would help to extract more information from our experiments, which helped me branch out into Data Science and Bioinformatics.

Leveraging my proficiency in Python and SQL, I've produced applications which control laboratory equipment, devised algorithms which help with experimental result analysis, and created full-stack web applications for data analysis which are deployed across multiple countries.
